Datagest
=

Datagest est un petit logiciel qui doit permettre de gérer une base de donnée facilement et en quelques clics.

Compatible
=
- Windows
- MacOS
- Linux

Features
=
 - Sauvegarder sa base de donnée
 - Se connecter à sa base de donnée
 - Déconnecter sa base de donnée
